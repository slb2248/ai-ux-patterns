# AI UX Patterns

Curated directory of **183 AI UX / AI design patterns** — chat, agents, trust, commerce, and more.

**[Browse interactive demos →](https://aiuxplayground.com/patterns)**

[![Patterns](https://img.shields.io/badge/patterns-183-blue)](https://aiuxplayground.com/patterns)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![Site](https://img.shields.io/badge/site-aiuxplayground.com-0A0A0A)](https://aiuxplayground.com)

Each pattern page here covers what it is, when to use it, and (where available) when not to, anti-patterns, and product examples. **Live demos and screenshots** are on [AI UX Playground](https://aiuxplayground.com).

- **[Streaming](https://aiuxplayground.com/pattern/streaming)** — Show replies token-by-token as they generate · [Notes](patterns/output/streaming.md)

- **[Confidence Score](https://aiuxplayground.com/pattern/confidence-score)** — Show how sure the model is about a claim · [Notes](patterns/trust/confidence-score.md)
