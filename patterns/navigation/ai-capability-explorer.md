> Interactive feature guide

## Overview

AI capability explorer is an interface design pattern that provides an interactive, searchable guide showing all available AI features and capabilities within an application. This UX pattern typically appears as a modal, sidebar, or dedicated page where users can browse, search, and learn about AI features organized by category or use case. Each capability includes descriptions, examples, and quick access buttons to try the feature. This pattern is essential for applications with extensive AI functionality where users need a comprehensive way to discover and understand available features. It serves as both a discovery tool and a learning resource, helping users understand the full scope of AI capabilities and how to apply them to their workflows.

## Good for

Perfect for AI-powered platforms, enterprise tools, and applications with extensive AI functionality where users need comprehensive feature discovery.
