**Category:** [Navigation](https://aiuxplayground.com/patterns/navigation)

> Semantic search across all features

## What it is

AI-powered search is an AI interface design pattern that provides semantic, intelligent search across all application features, content, and functionality, understanding user intent rather than just matching keywords. This UX pattern uses natural language processing to understand what users are looking for, even if they don't use exact terms. The search interface shows results from features, documents, conversations, and settings, ranked by relevance. Users can search for functionality, ask questions, or describe what they want to do. This pattern is essential for complex applications with many features where traditional search fails. It makes features discoverable and helps users find what they need through natural language.

## When to use

Perfect for complex applications, productivity tools, and platforms where semantic search makes features and content discoverable through natural language.
