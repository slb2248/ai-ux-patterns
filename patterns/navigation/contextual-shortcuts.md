**Category:** [Navigation](https://aiuxplayground.com/patterns/navigation)

> Shortcuts that adapt to current task

## What it is

Contextual shortcuts is an AI interface design pattern that provides keyboard shortcuts and quick actions that adapt to the user's current context, task, or location in the application. This UX pattern analyzes what the user is doing, where they are, and what they might want to do next, then surfaces relevant shortcuts or actions. Shortcuts change based on context, showing different options when editing text versus viewing content. The interface displays available shortcuts, explains what they do, and highlights contextually relevant ones. This pattern is essential for power users and productivity applications where context-aware shortcuts accelerate workflows.

## When to use

Perfect for productivity tools, power-user applications, and platforms where context-aware shortcuts improve workflow efficiency and discoverability.
