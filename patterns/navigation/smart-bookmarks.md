# Smart Bookmarks

**Category:** [Navigation](https://aiuxplayground.com/patterns/navigation)  
**Interactive demo:** [https://aiuxplayground.com/pattern/smart-bookmarks](https://aiuxplayground.com/pattern/smart-bookmarks)

> AI-suggested bookmarks based on usage

## What it is

Smart bookmarks is an AI interface design pattern that uses artificial intelligence to suggest relevant bookmarks, shortcuts, or quick links based on user behavior, patterns, and context. This UX pattern analyzes how users navigate the application, what they access frequently, and what they might need next, then suggests bookmarks or shortcuts that would be helpful. The interface shows suggested bookmarks with explanations, allows users to accept or dismiss suggestions, and learns from user preferences. This pattern is essential for complex applications where users benefit from personalized navigation shortcuts that adapt to their workflow patterns.

## When to use

Ideal for complex applications, productivity tools, and platforms where AI-suggested bookmarks improve navigation efficiency based on user behavior.

## Seen in

- Browser bookmarks
- Application shortcuts
- Dashboard widgets
- Quick access panels
