**Category:** [Outputs](https://aiuxplayground.com/patterns/output)

> Share outputs with permissions/links

## What it is

Output sharing is an AI interface design pattern that enables users to share AI-generated content with others through links, embeds, or exports, with configurable permissions and access controls. This UX pattern provides sharing options like public links, private shares with specific users, or embed codes for websites. Users can set permissions, expiration dates, and access levels. The interface shows sharing status, view counts, and allows users to revoke access. Shared outputs may include interactive previews or read-only views. This pattern is essential for collaboration, content distribution, and workflows where sharing AI outputs enables teamwork and knowledge sharing.

## When to use

Ideal for collaboration tools, content platforms, and workflows where sharing AI-generated outputs enables teamwork and knowledge distribution.
