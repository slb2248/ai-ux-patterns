> Choose output format (JSON, CSV, Markdown)

## Overview

Output format selection is an AI interface design pattern that allows users to choose the format of AI-generated content, such as JSON, CSV, Markdown, HTML, or plain text, with the AI adapting its output accordingly. This UX pattern provides format options before or after generation, allowing users to select the most appropriate format for their use case. The interface shows format descriptions, use cases, and previews, making it easy to choose. Users can switch formats without regenerating, and the AI maintains content quality across formats. This pattern is essential for data processing, content generation, and applications where output format directly impacts usability and integration.
