> Actions on select

## Overview

AI context menu is an interface design pattern that displays AI-powered actions in a contextual menu when users select text or content. This UX pattern provides relevant AI functions based on the selected content, such as "Explain this," "Summarize," "Translate," or "Rewrite," appearing in a tooltip or popup menu. The AI analyzes the selected content and suggests appropriate actions, making AI capabilities discoverable and contextually relevant. This pattern is essential for reading applications, browsers, and content platforms where users want to interact with text without leaving their current context. It makes AI features feel integrated and natural within the user's workflow.

## Good for

Ideal for reading applications, browsers, and content platforms where contextual AI actions enhance user interaction without disrupting the reading experience.
