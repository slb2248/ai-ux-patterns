**Category:** [Inputs](https://aiuxplayground.com/patterns/input)

> Draw or gesture to trigger AI actions

## What it is

Gesture input is an AI interface design pattern that enables users to trigger AI actions through drawing, sketching, or hand gestures, with computer vision interpreting the gestures as commands or inputs. This UX pattern allows users to draw shapes, symbols, or patterns that the AI recognizes and converts into actions like search queries, filters, or commands. The interface shows visual feedback as users draw, recognizes gestures in real-time, and provides suggestions for recognized patterns. This pattern is essential for touch interfaces, creative tools, and applications where gesture-based interaction is more intuitive than typing or clicking.

## When to use

Ideal for touch interfaces, creative applications, and mobile tools where gesture-based input provides intuitive, natural interaction methods.
