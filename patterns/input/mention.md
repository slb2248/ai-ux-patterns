# Context Mentions

**Category:** [Inputs](https://aiuxplayground.com/patterns/input)  
**Interactive demo:** [https://aiuxplayground.com/pattern/mention](https://aiuxplayground.com/pattern/mention)

> Reference files via @

## What it is

Context mentions is an AI interface design pattern that enables users to explicitly reference files, documents, data sources, or other AI agents by typing the @ symbol in chat interfaces. This UX pattern provides a discoverable way to pull external context into the conversation window, making AI interactions more powerful and contextually aware. When users type @, an autocomplete menu appears showing available files, documents, or agents they can mention, allowing them to build rich, multi-source conversations. This pattern is essential for enterprise AI tools and coding assistants where users need to reference specific code files, documentation, or knowledge bases to get accurate, context-aware responses.

## When to use

Perfect for AI coding assistants, enterprise knowledge management tools, and collaborative platforms where users need to reference specific files or documents in conversations.
