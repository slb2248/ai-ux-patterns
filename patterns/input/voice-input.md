# Voice Input

**Category:** [Inputs](https://aiuxplayground.com/patterns/input)  
**Interactive demo:** [https://aiuxplayground.com/pattern/voice-input](https://aiuxplayground.com/pattern/voice-input)

> Speech-to-text with visual feedback

## What it is

Voice input is an AI interface design pattern that enables users to interact with AI through speech, with real-time transcription and visual feedback showing listening state, transcription progress, and audio levels. This UX pattern makes AI accessible to users who prefer speaking over typing, are on mobile devices, or have accessibility needs. The interface provides clear visual indicators showing when the AI is listening, processing, or ready for input, with real-time transcription appearing as users speak. This pattern is essential for mobile applications, accessibility-focused tools, and hands-free interaction scenarios where voice input provides a more natural and convenient way to interact with AI.

## When to use

Essential for mobile applications, accessibility tools, and hands-free interaction scenarios where voice input provides a more natural and convenient interaction method.
