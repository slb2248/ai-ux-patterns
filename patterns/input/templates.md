> Starter prompts

## Overview

Prompt templates is an AI interface design pattern that provides pre-written, customizable prompt templates to help users get started with AI interactions. This UX pattern reduces the learning curve for new users by offering structured starting points for common tasks like writing emails, creating content, analyzing data, or generating code. Templates typically include placeholders that users can fill in, making it easy to customize prompts for specific needs. This pattern is essential for onboarding new users and improving prompt quality, as templates demonstrate best practices for interacting with AI. It's particularly valuable for content creation tools, writing assistants, and applications where users may not know how to effectively prompt AI.

## Good for

Ideal for content creation tools, writing assistants, and AI applications where providing starter prompts helps users discover capabilities and improve prompt quality.
