## Overview

Magic edit is an AI interface design pattern that allows users to select text and transform it using natural language instructions, with the AI rewriting the selection in place. This UX pattern provides inline editing capabilities where users can select text and describe how they want it changed, like "make it more formal," "shorten this," or "add examples", and the AI rewrites the selection accordingly. The pattern maintains context by working within the document flow, making edits feel natural and integrated. This pattern is essential for writing tools, content editors, and document applications where users need to refine and improve text without leaving their workflow.

## Good for

Perfect for writing tools, content editors, and document applications where inline text transformation improves the editing workflow.
