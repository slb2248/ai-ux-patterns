**Category:** [Inputs](https://aiuxplayground.com/patterns/input)

> Ghost text

## What it is

Predictive type is an AI interface design pattern that displays ghost text suggestions as users type, showing predicted sentence completions that users can accept with a single keystroke. This UX pattern uses AI to predict what users want to say next based on context, writing style, and common patterns, displaying suggestions in light gray text that users can accept by pressing Tab or continuing to type to ignore. This pattern accelerates writing by reducing typing effort, especially for repetitive or formulaic content like emails, messages, or forms. It's essential for communication tools, email clients, and text input applications where speed and convenience improve user experience.

## When to use

Perfect for email clients, messaging applications, and text input tools where predictive completions accelerate writing and reduce typing effort.
