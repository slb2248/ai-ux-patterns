> Time-based automation triggers

## Overview

Scheduled tasks and recurring actions is an AI interface design pattern that enables AI agents to execute tasks on schedules, with time-based triggers and recurring automation patterns. This UX pattern allows users to set up workflows that run automatically at specific times, intervals, or on recurring schedules like daily, weekly, or custom cron-like patterns. Users can configure schedules through intuitive time pickers, calendar interfaces, or natural language descriptions. The pattern provides visual feedback showing when tasks will run next and execution history. This pattern is essential for automation platforms, background job systems, and workflow tools where time-based triggers enable proactive automation without manual intervention.

## Good for

Perfect for automation platforms, background job systems, and workflow tools where scheduled and recurring tasks enable proactive, time-based automation.
