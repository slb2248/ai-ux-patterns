> Browse and install pre-built agents

## What it is

Agent marketplace is an AI interface design pattern that provides a browsable catalog of pre-built, configurable AI agents that users can discover, preview, and install. This UX pattern displays agents with descriptions, use cases, ratings, and previews, allowing users to search and filter by category, complexity, or use case. Users can see what each agent does, how it's configured, and try it before installing. The marketplace includes community-contributed agents, official templates, and verified agents. This pattern accelerates agent adoption by eliminating the need to build agents from scratch, making AI automation accessible to non-technical users.

## When to use

Perfect for agent platforms, automation tools, and ecosystems where providing pre-built agents accelerates adoption and reduces development time.
