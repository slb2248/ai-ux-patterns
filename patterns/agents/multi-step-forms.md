> Adaptive progressive forms

## What it is

Multi-step forms with AI is an AI interface design pattern that creates progressive form workflows where AI adapts subsequent steps based on previous answers, with smart field generation and dynamic form structure. This UX pattern uses AI to analyze user responses and intelligently determine what questions to ask next, which fields to show, and how to structure the remaining form steps. The AI can skip irrelevant sections, generate new fields based on context, and personalize the form experience for each user. This pattern reduces form abandonment by making forms feel shorter and more relevant, while ensuring all necessary information is collected. It's essential for lead generation, onboarding flows, and data collection applications where adaptive forms improve completion rates and user experience.

## When to use

Ideal for lead generation tools, onboarding flows, and data collection applications where adaptive, AI-powered forms improve completion rates and user experience.
