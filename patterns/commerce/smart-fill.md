> Unstructured to structured

## What it is

Smart fill is an AI interface design pattern that automatically parses unstructured text input, like addresses, names, or product information, and intelligently fills in structured form fields. This UX pattern reduces friction in checkout and data entry workflows by allowing users to paste or type information in natural formats, and the AI extracts and organizes it into the correct fields. For example, users can paste a full address and the AI separates it into street, city, state, and zip code fields. This pattern is essential for e-commerce checkout flows, form filling, and data entry applications where reducing user effort directly impacts conversion rates and user satisfaction.

## When to use

Perfect for e-commerce checkout flows, form filling applications, and data entry tools where parsing unstructured input reduces user effort and errors.
