> Context-aware product suggestions

## What it is

Smart recommendations is an AI interface design pattern that provides context-aware product suggestions based on user behavior, browsing history, preferences, and real-time context. This UX pattern uses machine learning to analyze user interactions, purchase patterns, and product relationships to deliver highly personalized recommendations. Unlike static suggestions, smart recommendations adapt to the user's current session, showing relevant products based on what they're viewing, items in their cart, or their past preferences. This pattern is essential for e-commerce platforms, content platforms, and discovery applications where personalized recommendations significantly impact engagement, conversion rates, and user satisfaction.

## When to use

Essential for e-commerce platforms, content discovery applications, and marketplaces where personalized recommendations drive engagement and sales.
