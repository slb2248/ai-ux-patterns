> Pin important messages

## What it is

Message pinning is an AI interface design pattern that allows users to pin important messages or AI responses within a conversation for quick reference. This UX pattern provides a visual indicator for pinned messages and often includes a dedicated section or sidebar showing all pinned items. Users can pin key insights, important code snippets, or critical information they want to reference later without scrolling through the entire conversation. This pattern is essential for long conversations where important information might get buried, making it easy to access key points quickly. It enhances productivity by reducing the time spent searching for important information.

## When to use

Ideal for long conversations, research sessions, and collaborative chats where users need quick access to key information.
